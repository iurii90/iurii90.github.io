RealPhotos

node.js, mySQL, HTML, CSS, JS, Bootstrap

Project for search photo albums by it's titles on vk.com network. There are no such function in the VK, so I made it by myself.

But it required to parse all users by API.

At first I created desktop parser application by C#. Information about albums and users was saved to mySQL databases.

Finally I got approximately 11 000 000 albums and 655 000 000 photos. Not files itself, but links to photos. The full project takes more than 140 gigabytes.

The main site made on node.js (Express) and Bootstrap.

There are three levels of access :

  • Guest (can search 10 times and can see only 50 first albums)
  • Low price account (infinity search and albums, but search conditions are restricted and it is impossible to see vk.com users whose albums there are)
  • High price account(infinity search with excessively conditions and you can see vk.com users)

Registration and payment system for accounts also working.
